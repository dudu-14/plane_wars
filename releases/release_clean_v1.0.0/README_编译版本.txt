本版本使用Nuitka编译器编译，具有以下优势：

- 🎯 更好的运行性能 - 运行时性能提升10-30%
- 📦 独立部署 - 无需安装Python环境
- 🛡️ 更好的兼容性 - 原生编译，兼容性更强

## 📁 文件说明

### 可执行文件
- `PlaneWars_Basic.exe` (384 KB) - 基础版本，需要Python环境
- `main.dist/PlaneWars_Standalone.exe` (18.8 MB) - 独立版本，推荐使用

### 推荐使用
**main.dist/PlaneWars_Standalone.exe** - 这是推荐的版本，因为：
- 完全独立，可以在任何Windows机器上运行
- 包含所有游戏依赖（pygame、numpy等）
- 性能最佳

## 🎮 使用方法

### 独立版本（推荐）
1. 进入 `main.dist` 文件夹
2. 双击运行 `PlaneWars_Standalone.exe`
3. 享受游戏！

### 基础版本
1. 确保系统已安装Python 3.x和相关依赖
2. 双击运行 `PlaneWars_Basic.exe`
